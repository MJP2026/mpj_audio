# mpj_audio
Here some AI based audio python code

## Generate virtual environment setup
python3 -m venv .venv

Mac:
source .venv/bin/activate

Windows:
.venv\Scripts\activate

## Install packages
pip install pyttsx3

python my_audio1_speaker.py
